## What is Sovereign Compute?

Sovereign compute is computation that is:

1. **Verifiable** — Formal proofs guarantee correctness
2. **Transparent** — Every step is visible and auditable
3. **Sealed** — Every execution leaves a WORM artifact
4. **Linear** — Resources are tracked and consumed exactly once
5. **Excavated** — Programs are revealed through execution
