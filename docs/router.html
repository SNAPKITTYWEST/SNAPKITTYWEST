Boot Sequence
1
Clone + Install
git clone https://github.com/SNAPKITTYWEST/SNAPKITTYWEST.git cd SNAPKITTYWEST && npm install
2
Configure AWS Bedrock
aws configure # region: us-east-1
3
Set .env
VAULT_MASTER_SECRET=your_secret MISTRAL_API_KEY=your_key # free at console.mistral.ai
4
Seal keys into vault
VAULT_MASTER_SECRET=your_secret npm run vault:seal "MISTRAL_API_KEY=your_key"
5
Route your first task
