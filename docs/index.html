Data Sources

CelesTrak
TLE · PUBLIC · NO KEY REQUIRED
Two-Line Element sets for active satellites. Fetched on satellite select. TLE age verified in the SIGNAL gate before any deed is issued.
N2YO REST API
REAL-TIME POSITION + PASS PREDICTIONS · FREE TIER: 100 REQ/DAY
Returns lat, lon, altitude for any NORAD ID. Also powers 7-day pass predictions with max elevation and duration. 30s LiveView interval keeps under rate limit.
n2yo_api_key: wired in config.exs
NASA GIBS
WMTS IMAGERY · PUBLIC · NO KEY REQUIRED
MODIS Terra true-color imagery for the CesiumJS globe layer. Selectable layer from 10+ available GIBS products. No authentication required.
NOAA GOES
ABI IMAGERY · PUBLIC · NO KEY REQUIRED
GOES-16/18 ABI full-disk weather imagery. Overlaid on the visual plane. Updated every 10 minutes from NOAA's public CDN.

Mesh Flow

01 · INGEST

Telemetry

CelesTrak TLE + N2YO position + NASA GIBS imagery + NOAA GOES weather.

02 · VERIFY

4 Gates

Source identity · schema shape · timestamp validity · freshness within 300s window.

03 · DEED

Trust Deed

Deed ID + deed hash generated from verified feed evidence. Signed by the Elixir runtime.

04 · SEAL

WORM Chain

Deed appended to JSONL chain on disk. SHA-256 hash. GenServer loads chain on boot.

05 · GOVERN

Agent Gate

Mesh events emitted. Agent actions allowed only after a sealed, verified, fresh deed.
